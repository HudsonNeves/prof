Módulo 3: Variáveis e Tipos de Dados

Aprenda como declarar variáveis e utilizar diferentes tipos de dados no JavaScript!

Variáveis e Tipos de Dados

Neste módulo, você aprenderá sobre variáveis, como declará-las e os principais tipos de dados disponíveis no JavaScript.

Declarando Variáveis

Existem três maneiras de declarar variáveis no JavaScript:

  • var: Forma mais antiga de declarar variáveis (não recomendada atualmente).
  • let: Forma moderna e recomendada para variáveis que podem ser reatribuídas.
  • const: Utilizada para variáveis que não podem ser reatribuídas.
<script>
  // Declarando variáveis
  var idade = 25;
  let nome = "Maria";
  const PI = 3.14159;

  console.log(idade); // Exibe: 25
  console.log(nome); // Exibe: Maria
  console.log(PI); // Exibe: 3.14159
</script>

Tipos de Dados

O JavaScript possui vários tipos de dados. Os principais são:

  • String: Cadeia de caracteres, como "Olá, mundo!".
  • Number: Números, inteiros ou decimais, como 42 ou 3.14.
  • Boolean: Verdadeiro ou falso (true ou false).
  • Null: Representa ausência de valor.
  • Undefined: Variável declarada, mas sem valor atribuído.
  • Object: Estruturas mais complexas, como arrays e objetos.
<script>
  // Exemplos de tipos de dados
  let texto = "JavaScript"; // String
  let numero = 123; // Number
  let isTrue = true; // Boolean
  let vazio = null; // Null
  let naoDefinido; // Undefined
</script>

Exercício Prático

1. Crie uma página HTML e adicione o seguinte código em uma tag <script>:

<script>
  let idade = 30;
  let nome = "João";
  console.log("Meu nome é " + nome + " e tenho " + idade + " anos.");
</script>

2. Abra a página no navegador e veja o resultado no console (pressione F12 para abrir o console do desenvolvedor).
