Módulo 5: Estruturas Condicionais

Aprenda a tomar decisões no seu código utilizando estruturas condicionais no JavaScript!

O que são Estruturas Condicionais?

Estruturas condicionais permitem executar diferentes blocos de código dependendo de uma condição. Isso é essencial para criar programas dinâmicos e interativos.

If...Else

O if avalia uma condição. Se ela for true, o bloco de código será executado. O else é executado quando a condição é false.

<script>
  let idade = 18;

  if (idade >= 18) {
    console.log("Você é maior de idade.");
  } else {
    console.log("Você é menor de idade.");
  }
</script>

If...Else If...Else

Use else if para testar múltiplas condições antes de executar o bloco else.

<script>
  let nota = 85;

  if (nota >= 90) {
    console.log("Nota: A");
  } else if (nota >= 80) {
    console.log("Nota: B");
  } else {
    console.log("Nota: C ou menor");
  }
</script>

Switch

O switch é uma alternativa ao if...else quando você precisa comparar o mesmo valor com várias condições.

<script>
  let dia = "terça";

  switch (dia) {
    case "segunda":
      console.log("Hoje é segunda-feira.");
      break;
    case "terça":
      console.log("Hoje é terça-feira.");
      break;
    default:
      console.log("Dia não reconhecido.");
  }
</script>

Exercício Prático

Crie um script que avalie a idade de uma pessoa e retorne:

  1. "Criança" se a idade for menor que 12;
  2. "Adolescente" se a idade estiver entre 12 e 17;
  3. "Adulto" se a idade for maior ou igual a 18.
<script>
  let idade = 15;

  if (idade < 12) {
    console.log("Criança");
  } else if (idade < 18) {
    console.log("Adolescente");
  } else {
    console.log("Adulto");
  }
</script>
