Módulo 4: Operadores Aritméticos e Comparações

Entenda como realizar operações matemáticas e comparações no JavaScript!

Operadores Aritméticos

Os operadores aritméticos permitem realizar cálculos matemáticos em variáveis e valores. Alguns dos principais operadores incluem:

  • +: Adição
  • -: Subtração
  • *: Multiplicação
  • /: Divisão
  • %: Resto da divisão
  • **: Exponenciação
<script>
  // Exemplos de operadores aritméticos
  let a = 10;
  let b = 3;

  console.log(a + b); // Adição: 13
  console.log(a - b); // Subtração: 7
  console.log(a * b); // Multiplicação: 30
  console.log(a / b); // Divisão: 3.3333
  console.log(a % b); // Resto: 1
  console.log(a ** b); // Exponenciação: 1000
</script>

Operadores de Comparação

Os operadores de comparação retornam valores booleanos (true ou false) ao comparar dois valores:

  • ==: Igualdade de valor
  • ===: Igualdade estrita (valor e tipo)
  • !=: Diferente
  • !==: Diferente estrito (valor e tipo)
  • >: Maior que
  • <: Menor que
  • >=: Maior ou igual
  • <=: Menor ou igual
<script>
  // Exemplos de operadores de comparação
  let x = 5;
  let y = "5";

  console.log(x == y); // true (apenas valor é comparado)
  console.log(x === y); // false (valor e tipo são comparados)
  console.log(x != y); // false (valor é igual)
  console.log(x !== y); // true (tipo é diferente)
</script>

Exercício Prático

1. Crie uma página HTML e insira o seguinte código dentro de uma tag <script>:

<script>
  let num1 = 15;
  let num2 = 4;

  console.log("Soma: " + (num1 + num2));
  console.log("Maior que 10: " + (num1 > 10));
</script>

2. Abra a página no navegador e veja o resultado no console (F12 para abrir o console do desenvolvedor).
