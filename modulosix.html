Módulo 6: Estruturas de Repetição

Aprenda a executar ações repetitivas no seu código de forma eficiente utilizando laços de repetição.

O que são Estruturas de Repetição?

As estruturas de repetição, ou laços, permitem executar um bloco de código várias vezes com base em uma condição ou conjunto de iterações.

While

O laço while continua a executar enquanto a condição for verdadeira.

<script>
  let contador = 1;

  while (contador <= 5) {
    console.log("Contagem: " + contador);
    contador++;
  }
</script>

Do...While

O laço do...while executa o bloco ao menos uma vez antes de verificar a condição.

<script>
  let numero = 1;

  do {
    console.log("Número: " + numero);
    numero++;
  } while (numero <= 3);
</script>

For

O laço for é usado quando você sabe o número de iterações com antecedência.

<script>
  for (let i = 1; i <= 5; i++) {
    console.log("Iteração: " + i);
  }
</script>

For...Of

O laço for...of é usado para iterar sobre elementos de arrays ou strings.

<script>
  let frutas = ["maçã", "banana", "laranja"];

  for (let fruta of frutas) {
    console.log("Fruta: " + fruta);
  }
</script>

Exercício Prático

Crie um script que exiba os números pares de 1 a 10 usando cada uma das estruturas de repetição acima.

<script>
  console.log("Usando for:");
  for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) console.log(i);
  }

  console.log("Usando while:");
  let j = 1;
  while (j <= 10) {
    if (j % 2 === 0) console.log(j);
    j++;
  }

  console.log("Usando do...while:");
  let k = 1;
  do {
    if (k % 2 === 0) console.log(k);
    k++;
  } while (k <= 10);
</script>
