Módulo 2: Inserir JavaScript

Aprenda como inserir JavaScript em páginas web de maneira eficaz!

Como Inserir JavaScript

Neste módulo, você aprenderá como incluir JavaScript em páginas web, tanto internamente quanto externamente, para organizar melhor o código e torná-lo reutilizável.

Métodos para inserir JavaScript

  1. Script interno: O código JavaScript é inserido diretamente no arquivo HTML, dentro da tag <script>.
    2. <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>JavaScript Interno</title>
    </head>
    <body>
      <script>
        alert("Olá, mundo! Este é um script interno.");
      </script>
    </body>
    </html>
  2. Script externo: O código JavaScript é armazenado em um arquivo separado com extensão .js e referenciado no HTML usando a tag <script>.
    3. // Arquivo: script.js
    alert("Olá, mundo! Este é um script externo.");

    // HTML
    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>JavaScript Externo</title>
      <script src="script.js"></script>
    </head>
    <body>
    </body>
    </html>

Exercício Prático

1. Crie um arquivo chamado script.js e adicione nele a seguinte linha:

alert("Bem-vindo ao curso de JavaScript!");

2. Referencie este arquivo em um documento HTML com a tag <script>, como mostrado acima.

3. Abra o arquivo HTML no navegador e verifique se o alerta é exibido corretamente.

Resultado Esperado

Ao abrir o arquivo HTML no navegador, um alerta será exibido com a mensagem configurada no arquivo script.js. Isso demonstra como o JavaScript pode ser organizado de forma eficiente.
