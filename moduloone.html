Módulo 1: Primeiro Programa

Aprenda a criar seu primeiro programa em JavaScript.

Introdução ao Módulo

O JavaScript é uma linguagem poderosa e versátil. Ele é usado para adicionar interatividade e dinamismo a sites. Antes de começar, você precisará de um navegador web (como o Google Chrome) e um editor de texto ou IDE (como o Visual Studio Code).

Passos para criar seu primeiro programa:

  1. Abra seu editor de texto e crie um novo arquivo chamado primeiro-programa.html.
  2. Adicione a estrutura básica de um documento HTML:
    3. <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Primeiro Programa</title>
</head>
<body>
    <script>
        // Seu primeiro código JavaScript
        console.log("Olá, mundo!");
    </script>
</body>
</html>
  3. Salve o arquivo.
  4. Abra o arquivo em um navegador clicando duas vezes sobre ele ou arrastando para o navegador.
  5. Pressione F12 ou clique com o botão direito na página e selecione Inspecionar. Navegue até a aba Console para visualizar a mensagem Olá, mundo!.

Parabéns! Você acabou de executar seu primeiro programa em JavaScript. A partir daqui, você começará a construir funcionalidades mais complexas e úteis.

Exercício Prático

1. Modifique o código acima para exibir sua mensagem personalizada no console, por exemplo, console.log("Bem-vindo ao curso de JavaScript!");.

2. Adicione mais um console.log que calcule e exiba a soma de dois números, como no exemplo: 

console.log("A soma de 5 + 3 é: " + (5 + 3));

3. Teste o código no navegador para verificar os resultados.

Resultado Esperado

No console do navegador, você verá as mensagens exibidas sequencialmente, incluindo a soma dos números. Isso mostra como o JavaScript processa e exibe informações.
