Objetivo da Aula

Na aula de hoje, vamos aprender sobre as estruturas condicionais, fundamentais para criar decisões dentro de um algoritmo. Você vai entender como usar condições para que seu programa tome decisões com base em determinados critérios.

O que são Estruturas Condicionais?

Estruturas condicionais permitem que seu programa tome decisões baseadas em certas condições, como "Se isso acontecer, então faça aquilo". Essas estruturas são essenciais para a criação de algoritmos dinâmicos que respondem a diferentes situações.

Existem dois tipos principais de estruturas condicionais que veremos: se (if) e se-senão (else). Elas ajudam o programa a decidir entre dois ou mais caminhos possíveis.

Exemplo Prático de Estrutura Condicional

Vamos ver um exemplo simples utilizando o Visualg para entender como as estruturas condicionais funcionam:

Algoritmo "Exemplo_Condicional"
    // Curso       : Técnico em Desenvolvimento de Sistemas
    // Professor   : Hudson Neves
    // Descrição   : Exemplo de estrutura condicional

Var
    idade : Inteiro

Inicio
    escreva("Digite sua idade: ")
    leia(idade)
    
    se idade >= 18 então
        escreval("Você é maior de idade!")
    senao
        escreval("Você é menor de idade!")
    fimse
Fimalgoritmo

Este código verifica se o usuário tem 18 anos ou mais. Se sim, o programa informa que ele é maior de idade. Caso contrário, informa que ele é menor de idade. Esse é um exemplo básico de uma estrutura condicional se-senão.

Por que as Estruturas Condicionais São Importantes?

As estruturas condicionais são essenciais porque permitem que um programa reaja de maneira diferente a situações distintas. Elas são a base de lógica de controle de fluxo e são amplamente usadas em sistemas que precisam de decisões dinâmicas, como jogos, simuladores e sistemas financeiros.

Dominar o uso de condições vai permitir que você crie algoritmos mais inteligentes e interativos.
